# Settings

Open **Settings -> Community plugins -> New Note Fixer** to configure the plugin. The option below lists the setting key stored in the plugin's `data.json`.

## Folder location

- `shouldPromptForFolderLocation` - when enabled, clicking a link to a non-existing note opens a folder picker so you choose where the new note is created, instead of New Note Fixer placing it automatically under your **Default location for new notes**. The picker also lets you type a new folder name to create it on the spot. When disabled (the default), the note is placed automatically and out-of-vault relative links (`[[../folder/note]]`) are refused with a notice - see [[02 Relative link fix]].
